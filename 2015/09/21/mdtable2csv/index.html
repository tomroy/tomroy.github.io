mdtable2csv

Recently I was trying to find a easy way to convert the tables in md to excel, instead of just copy paste.
After some research, I found that there are so many converters on the internet, but none of them can convert markdown file to excel. So I guess I have to make one on my own. And I was just happen to be learning Python, so I decided to write a Python! I also put it in my GitHub repository.

After lots of search on the internet, I found that there’s a GitHub api that can convert your markdown to html. So I can send my markdown file using json format to github, and it’ll return the html format to me.

After I got the html, I first find all the contents in the <table>...</table> tags, extract the rows in the HTML table from <tr> tag. Then get the Header cells <th> which contains the header information and Standard cells <td> which contains data
separate headers and data by delimiter ,.
