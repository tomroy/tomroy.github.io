Markdown tips

半形空白(Halfwidth space) = &ensp; 或 &#8194;
全形空白(Fullwidth space) = &emsp; 或 &#8195;
不斷行的空白格(non-breaking space) = &nbsp; 或 &#160;

半形空白(Halfwidth space)

執行結果如下：

哈囉！這邊     有五個半形空白。
Hello! We have     5 halfwidth spaces here.

全形空白(Fullwidth space)

執行結果如下：

哈囉！這邊     有五個全形空白。
Hello! We have     5 fullwidth spaces here.

不斷行的空白格(non-breaking space)

執行結果如下：

哈囉！這邊     有五個不斷行的空白格。
Hello! We have     5 non-breaking spaces here.
